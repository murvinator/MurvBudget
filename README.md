# MurvBudget

A clean and fast personal budget app built as a Progressive Web App (PWA) for iPhone. No account required, no bank connections — just a simple, private way to track your spending.

**[Open the app →](https://murvbudgetapp.netlify.app/)**

## Features

- **Visual charts** — understand your spending patterns at a glance
- **Monthly breakdown** — track income and expenses month by month
- **Debt tracking** — log and manage debts with payment history
- **Budget goals** — set spending limits per category
- **Private by default** — no account, no tracking, data lives on your device
- **Offline ready** — works without an internet connection
- **Install to home screen** — native app experience on iPhone via Safari

---

## Install on iPhone

1. Open [murvbudgetapp.netlify.app](https://murvbudgetapp.netlify.app/) in **Safari**
2. Tap the **Share** button at the bottom of the screen
3. Tap **Add to Home Screen**
4. Tap **Add** — done!

## Tech Stack

| | |
|---|---|
| Framework | [Vue 3](https://vuejs.org/) (Composition API) |
| Build tool | [Vite](https://vitejs.dev/) |
| State management | [Pinia](https://pinia.vuejs.org/) with persisted state |
| Deployment | [Netlify](https://www.netlify.com/) |
